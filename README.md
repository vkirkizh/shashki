# Russian Draughts (2011)

A Russian draughts (shashki) game with a computer player.

This project was developed as my bachelor's thesis in 2011 while I was studying at Saint Petersburg State University of Aerospace Instrumentation.

## Features

- Full game logic written in C++
- GUI built with Qt4
- Support for Linux and Maemo
- Play against another human or the computer
- Computer player based on minimax, alpha-beta pruning and other search algorithms
- Multithreaded search implementation
- Tournament system for benchmarking different computer player implementations
- Unit tests
